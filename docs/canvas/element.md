## graph.addItem 
设置单个元素
```javascript
graph.addItem(type: 'node' | 'edge', data: Model)
```

## graph.addNode
添加单个节点元素
```javascript
graph.addNode(data: Model)
```

## graph.addEdge
添加单个线元素
```javascript
graph.addEdge(data: Model)
```

## graph.updateNodeData
更新某个单个节点元素,data中的id是必填。
```javascript
graph.addEdge(data: Model)
```
